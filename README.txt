Para poder generar el ejecutable de este proyecto usted debe
ir al directorio code/ en el cual se encuentra el archivo make.
----------------------------------------------------------------------
Dentro del directorio code/ se encuentra el directorio src/
el cual contiene el codigo main en c++ y su respectivo  archivo makefile
Este tambien se encuentran el archivo global y el directorio include/ 
el cual contiene los include a utilizar.
---------------------------------------------------------------------
Al realizar el comando make el ejecutable se llamara test02
para poder utilizarlo se debe escribir :

./test02
----------------------------------------------------------------------
Tambien si quiere definir el numero de repeticiones y tamaño del problema
puede realizarlo de la forma:

./test02 -nt [numero de iteraciones] -nx [tamaño del problema]
